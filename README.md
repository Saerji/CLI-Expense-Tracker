# CLI Expense Tracker

## Project Overview
A CLI Expense Tracker Project that utilizes Python in order to make a project that allow users to be able to manage informations regarding expenses. A project made in compliance to the [CLI Expense Tracker Project](https://roadmap.sh/projects/expense-tracker) provided by [roadmap.sh](https://roadmap.sh) for the [Backend Development Roadmap](https://roadmap.sh/backend).

## Features
➣ **Adding** expense information.  
➣ **Viewing** all expense informations.  
➣ **Update** existing expense informations one by one.
➣ **Delete** existing expenses that may no longer be needed.
➣ **Summarize** existing expenses information by the month they're added.

## Usage
```
python main.py add # add expense info
python main.py view # view all expenses
python main.py summary # display summary of expenses
python main.py delete # delete existing expenses
python main.py update # update existing expenses
```
